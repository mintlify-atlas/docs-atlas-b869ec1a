## Component Categories

### Plan Display Components

These components help you present pricing plans to your users:

- **HorizontalPlanCards** - Full-width plan cards stacked vertically, ideal for detailed feature comparisons
- **VerticalPlanCards** - Traditional side-by-side plan cards in a grid layout
- **HighlightedPlanCards** - Premium card design with visual emphasis on recommended plans

### Subscription Management

- **SubscriptionSummaryCard** - Display and manage active subscriptions with built-in cancellation flow
- **SubscriptionSummaryPage** - Full-page wrapper for subscription management with automatic Mantle integration

### Engagement Components

- **ChecklistCard** - Interactive checklists for onboarding and feature adoption
- **BundleCard** - Promotional component for app bundles and cross-selling

## Common Features

All plan card components share these features:

- Monthly/yearly interval toggle
- Recommended plan badges
- Custom CTA buttons
- Trial period display
- Discount indicators
- Custom plan support
- Shopify Polaris integration
- Responsive design

## Getting Started

All components are exported from the main package:

```jsx
import {
  HorizontalPlanCards,
  VerticalPlanCards,
  HighlightedPlanCards,
  SubscriptionSummaryCard,
  SubscriptionSummaryPage,
  ChecklistCard,
  BundleCard,
} from '@heymantle/polaris';
```

<Note>
  All components require the Mantle customer and plans data from the `@heymantle/react` hooks. See the [Installation](/installation) and [Quickstart](/quickstart) guides for setup instructions.
</Note>

## Design Philosophy

Mantle Polaris components are designed with these principles:

1. **Conversion-Optimized** - Layouts and designs proven to maximize subscription conversions
2. **Shopify Native** - Built with Polaris components for seamless integration
3. **Customizable** - Extensive props for customization without losing best practices
4. **Accessible** - WCAG compliant and keyboard navigable
5. **Mobile-First** - Responsive designs that work on all devices
