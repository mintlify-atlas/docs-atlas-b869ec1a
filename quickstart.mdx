This guide walks you through creating a complete pricing page with plan selection and subscription handling.

## What You'll Build

By the end of this guide, you'll have:

- A pricing page displaying your Mantle plans
- Plan selection with monthly/yearly toggle
- Subscription flow integration
- Current subscription display

## Prerequisites

<Steps>
  <Step title="Install Dependencies">
    Make sure you have all required packages installed:

    ```bash
    npm install @heymantle/polaris @heymantle/react @shopify/polaris
    ```

    See the [installation guide](/installation) for detailed instructions.
  </Step>

  <Step title="Get Mantle Credentials">
    You'll need:
    - Your Mantle App ID
    - A customer API token (obtained server-side via the Mantle identify endpoint)

    <Note>
    Customer tokens are generated server-side after identifying a customer with Mantle. See [Mantle's documentation](https://heymantle.com/docs) for details.
    </Note>
  </Step>

  <Step title="Configure Plans">
    Create at least one plan in your [Mantle dashboard](https://heymantle.com/dashboard) before proceeding.
  </Step>
</Steps>

## Step 1: Setup MantleProvider

Wrap your app with `MantleProvider` near the root of your component tree. This establishes the connection to Mantle's API.

```jsx App.jsx
import { useState, useEffect } from "react";
import { AppProvider } from "@shopify/polaris";
import { MantleProvider } from "@heymantle/react";
import "@shopify/polaris/build/esm/styles.css";
import PricingPage from "./PricingPage";

function App() {
  const [shop, setShop] = useState(null);

  // Fetch shop data including Mantle customer token
  useEffect(() => {
    async function fetchShop() {
      const response = await fetch("/api/shop");
      const data = await response.json();
      setShop(data);
    }
    fetchShop();
  }, []);

  if (!shop) {
    return <div>Loading...</div>;
  }

  return (
    <AppProvider>
      <MantleProvider
        appId={process.env.MANTLE_APP_ID}
        customerApiToken={shop.mantleApiToken}
        apiUrl={process.env.MANTLE_API_URL} // Optional: defaults to production
      >
        <PricingPage />
      </MantleProvider>
    </AppProvider>
  );
}

export default App;
```

<Warning>
Never expose your Mantle API token in client-side code. Only use the customer-specific token obtained from your backend.
</Warning>

## Step 2: Display Plans with HorizontalPlanCards

Create a pricing page component that displays your plans using the `HorizontalPlanCards` component.

```jsx PricingPage.jsx
import { useMantle } from "@heymantle/react";
import { HorizontalPlanCards } from "@heymantle/polaris";

function PricingPage() {
  const { customer, plans, subscribe } = useMantle();

  const handleSubscribe = async ({ planId, discountId }) => {
    try {
      // Call Mantle's subscribe function
      const subscription = await subscribe({ planId, discountId });
      
      // Redirect to Shopify's confirmation page
      window.open(subscription.confirmationUrl, "_top");
    } catch (error) {
      console.error("Subscription failed:", error);
    }
  };

  return (
    <HorizontalPlanCards
      customer={customer}
      plans={plans}
      onSubscribe={handleSubscribe}
      showPlanIntervalToggle={true}
      showTrialDaysAsFeature={true}
      showRecommendedBadge={true}
    />
  );
}

export default PricingPage;
```

## Step 3: Add Subscription Management

Show current subscription details and allow users to manage their plan using the `SubscriptionSummaryCard` component.

```jsx AccountPage.jsx
import { useMantle } from "@heymantle/react";
import { Page, Layout } from "@shopify/polaris";
import { SubscriptionSummaryCard } from "@heymantle/polaris";
import { useNavigate } from "react-router-dom";

function AccountPage() {
  const { customer, subscription, cancelSubscription, refetch } = useMantle();
  const navigate = useNavigate();

  return (
    <Page title="Account Settings">
      <Layout>
        <Layout.Section>
          <SubscriptionSummaryCard
            subscription={subscription}
            cancelSubscription={cancelSubscription}
            refetch={refetch}
            onShowPlans={() => navigate("/pricing")}
            onPlanCancelled={() => {
              console.log("Subscription cancelled");
            }}
          />
        </Layout.Section>
      </Layout>
    </Page>
  );
}

export default AccountPage;
```

## Step 4: Handle Subscription Confirmation

After a user subscribes, Shopify redirects back to your app with a `?subscribed=true` query parameter. The `HorizontalPlanCards` component automatically shows a success banner when this parameter is present.

```jsx
// The component handles this automatically:
<HorizontalPlanCards
  customer={customer}
  plans={plans}
  onSubscribe={handleSubscribe}
  // Success banner appears when ?subscribed=true is in URL
/>
```

## Component Props Reference

### HorizontalPlanCards Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `customer` | `Customer` | Required | Customer object from `useMantle()` |
| `plans` | `Plan[]` | Required | Array of plans from `useMantle()` |
| `onSubscribe` | `Function` | Required | Callback when user selects a plan |
| `showPlanIntervalToggle` | `boolean` | `true` | Show monthly/yearly toggle |
| `showTrialDaysAsFeature` | `boolean` | `true` | Display trial period as a feature |
| `showRecommendedBadge` | `boolean` | `true` | Show badge on recommended plans |
| `useShortFormPlanIntervals` | `boolean` | `false` | Use "mo" instead of "month" |
| `pageWidth` | `string` | `"default"` | `"default"`, `"narrow"`, or `"full"` |
| `backUrl` | `string` | `""` | URL for back button (empty hides it) |
| `customFieldCta` | `string` | `undefined` | Custom field key for CTA button text |

### SubscriptionSummaryCard Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `subscription` | `Subscription` | Required | Current subscription object |
| `onShowPlans` | `Function` | Required | Callback to navigate to pricing |
| `cancelSubscription` | `Function` | Required | Function to cancel subscription |
| `refetch` | `Function` | Optional | Refetch customer data after changes |
| `onPlanCancelled` | `Function` | Optional | Callback after cancellation |
| `orientation` | `string` | `"horizontal"` | `"horizontal"` or `"vertical"` |
| `hideSelectPlanButton` | `boolean` | `false` | Hide the "Select Plan" button |

For detailed feature lists, use `VerticalPlanCards` with stacked layouts:

```jsx
import { VerticalPlanCards } from "@heymantle/polaris";

<VerticalPlanCards
  customer={customer}
  plans={plans}
  onSubscribe={handleSubscribe}
  trialDaysAsFeature={true}
  useShortFormPlanIntervals={true}
/>

<HighlightedPlanCards

  plans={plans}
  onSubscribe={handleSubscribe}
  customFieldPlanRecommended="Recommended"
  showRecommendedBadge={true}
/>
```

## Using the useMantle Hook

The `useMantle` hook provides access to Mantle data and operations throughout your component tree:

```jsx
import { useMantle } from "@heymantle/react";

function MyComponent() {
  const {
    customer,        // Current customer object
    subscription,    // Active subscription (if any)
    plans,          // Available plans for customer
    subscribe,      // Function to create subscription
    cancelSubscription, // Function to cancel subscription
    pushEvent       // Function to track custom events
  } = useMantle();

  // Track a page view
  useEffect(() => {
    pushEvent({
      eventName: 'page_view',
      properties: { path: window.location.pathname }
    });
  }, []);

  return (
    <div>
      {subscription ? (
        <p>Current plan: {subscription.plan.name}</p>
      ) : (
        <p>No active subscription</p>
      )}
    </div>
  );
}
```

## Complete Example

Here's a complete example combining all the concepts:

```jsx Complete App
import { useState, useEffect } from "react";
import { BrowserRouter, Routes, Route } from "react-router-dom";
import { AppProvider } from "@shopify/polaris";
import { MantleProvider, useMantle } from "@heymantle/react";
import { HorizontalPlanCards, SubscriptionSummaryCard } from "@heymantle/polaris";
import "@shopify/polaris/build/esm/styles.css";

function PricingPage() {
  const { customer, plans, subscribe } = useMantle();

  const handleSubscribe = async ({ planId, discountId }) => {
    const subscription = await subscribe({ planId, discountId });
    window.open(subscription.confirmationUrl, "_top");
  };

  return (
    <HorizontalPlanCards
      customer={customer}
      plans={plans}
      onSubscribe={handleSubscribe}
      showPlanIntervalToggle={true}
      showRecommendedBadge={true}
    />
  );
}

function AccountPage() {
  const { subscription, cancelSubscription, refetch } = useMantle();

  return (
    <SubscriptionSummaryCard
      subscription={subscription}
      cancelSubscription={cancelSubscription}
      refetch={refetch}
      onShowPlans={() => window.location.href = "/pricing"}
    />
  );
}

function App() {
  const [shop, setShop] = useState(null);

  useEffect(() => {
    fetch("/api/shop")
      .then(res => res.json())
      .then(setShop);
  }, []);

  if (!shop) return <div>Loading...</div>;

  return (
    <AppProvider>
      <MantleProvider
        appId={process.env.MANTLE_APP_ID}
        customerApiToken={shop.mantleApiToken}
      >
        <BrowserRouter>
          <Routes>
            <Route path="/pricing" element={<PricingPage />} />
            <Route path="/account" element={<AccountPage />} />
          </Routes>
        </BrowserRouter>
      </MantleProvider>
    </AppProvider>
  );
}

export default App;
```
