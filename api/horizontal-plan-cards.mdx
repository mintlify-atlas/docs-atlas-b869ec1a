---
title: "HorizontalPlanCards"
description: "Display plan selection UI with horizontally-oriented plan cards"
---

The `HorizontalPlanCards` component renders a full-page plan selection interface with plans displayed in horizontal card layouts. It includes support for monthly/yearly toggles, custom plans, and automatic success banners.

## Usage

```jsx
import { HorizontalPlanCards } from '@heymantle/polaris';

function PlanSelectionPage() {
  return (
    <HorizontalPlanCards
      customer={customer}
      plans={plans}
      onSubscribe={(plan) => {
        console.log('Selected plan:', plan);
        // Handle subscription logic
      }}
      backUrl="/dashboard"
      showRecommendedBadge={true}
      customFieldCta="buttonLabel"
      customFieldPlanRecommended="Recommended"
      showPlanIntervalToggle={true}
      showTrialDaysAsFeature={true}
      useShortFormPlanIntervals={true}
      pageWidth="default"
      showCustomPlans={true}
      toggleYearlySubtitle={false}
      showFeatureDescriptionAsTooltip={true}
    />
  );
}
```

## Props

<ParamField path="customer" type="object" required>
  Customer object containing subscription information. Used to determine the current plan and display appropriate UI states.
  
  Expected structure:
  ```javascript
  {
    subscription: {
      plan: { id: string },
      active: boolean,
      cancelOn: string | null
    }
  }
  ```
</ParamField>

<ParamField path="plans" type="array" required>
  Array of plan objects to display. Each plan should include pricing, features, and metadata.
  
  Expected structure:
  ```javascript
  [{
    id: string,
    name: string,
    interval: "EVERY_30_DAYS" | "ANNUAL",
    availability: "public" | "customer" | "customerTag",
    amount: number,
    currencyCode: string,
    customFields: object,
    features: array,
    trialDays: number
  }]
  ```
</ParamField>

<ParamField path="onSubscribe" type="function" required>
  Callback function triggered when a user selects a plan. Receives the selected plan object as an argument.
  
  ```javascript
  (plan) => void
  ```
</ParamField>

<ParamField path="backUrl" type="string" default="">
  URL for the back button in the page header. If empty string or null, the back button will be hidden.
</ParamField>

<ParamField path="showRecommendedBadge" type="boolean" default={true}>
  Whether to display a "Recommended" badge on plans marked as recommended via custom fields.
</ParamField>

<ParamField path="customFieldCta" type="string">
  The key name in plan's `customFields` object to use for custom button labels. For example, if set to `"cta"`, the component will look for `plan.customFields.cta` to override the default button text.
</ParamField>

<ParamField path="customFieldPlanRecommended" type="string" default="Recommended">
  The key name in plan's `customFields` object to identify recommended plans. Plans with this custom field will display the recommended badge (if `showRecommendedBadge` is true).
</ParamField>

<ParamField path="showPlanIntervalToggle" type="boolean" default={true}>
  Whether to show the Monthly/Yearly toggle in the page header. Only appears if both monthly and yearly plans are available.
</ParamField>

<ParamField path="showTrialDaysAsFeature" type="boolean" default={true}>
  Whether to display trial days as a feature in the plan's feature list (e.g., "14-day free trial").
</ParamField>

<ParamField path="useShortFormPlanIntervals" type="boolean">
  Use shortened interval labels (e.g., "$/mo" instead of "$/month", "$/yr" instead of "$/year").
</ParamField>

<ParamField path="pageWidth" type="string" default="default">
  Controls the page width. Options:
  - `"default"`: Standard Polaris page width
  - `"narrow"`: Narrow page width
  - `"full"`: Full-width page
</ParamField>

<ParamField path="showCustomPlans" type="boolean" default={true}>
  Whether to display custom plans (plans with availability other than "public") in a separate section below public plans.
</ParamField>

<ParamField path="toggleYearlySubtitle" type="boolean" default={false}>
  For annual plans, show the monthly equivalent price as a subtitle (e.g., "$120/year" with "$10/month" shown below).
</ParamField>

<ParamField path="showFeatureDescriptionAsTooltip" type="boolean" default={true}>
  Display feature descriptions in tooltips when hovering over feature names, rather than inline.
</ParamField>

## Features

### Automatic Success Banner

The component automatically displays a success banner when the URL contains `?subscribed=true`, indicating a successful subscription. Users can dismiss the banner, which removes the query parameter from the URL.

### Plan Interval Toggle

When both monthly and annual plans are available and `showPlanIntervalToggle={true}`, a segmented button group appears in the page header allowing users to switch between viewing monthly and yearly plans.

### Custom Plans Section

Plans with `availability` set to anything other than `"public"` (such as `"customer"` or `"customerTag"`) are displayed in a separate "Custom Plans" section below the public plans, separated by a divider.

### Responsive Layout

The horizontal card layout automatically adapts to different screen sizes, stacking cards vertically on mobile devices.
